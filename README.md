# 📝 Sticky Notes App

A simple and lightweight sticky notes application that lets users quickly create, edit, and delete notes. Perfect for keeping track of ideas, tasks, and reminders.

## 🚀 Features

* Create new sticky notes
* Edit existing notes
* Delete notes
* Persistent storage (notes saved even after refresh)
* Clean and minimal UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 💡 Usage

* Click on **"Add Note"** to create a new sticky note
* Type your content directly into the note
* Use the delete button to remove a note
